Quem sou eu

Olá! Sou Vithor Roder, desenvolvedor Full-Stack baseado no Rio de Janeiro 🇧🇷. Construo aplicações web e desktop com foco em arquitetura limpa e código de qualidade.

Minha linguagem principal é Java — uso no dia a dia tanto para backend (Spring Boot, Quarkus, Hibernate, JPA, Flyway, OkHttp, JUnit) quanto para desktop (Swing, JavaFX, SWT). No frontend web trabalho com HTML, CSS, JavaScript e Tailwind/Windi.

Sou desenvolvedor autodidata com experiência prática em projetos reais. Sempre explorando novas tecnologias e aprimorando minhas skills. Foco em arquitetura limpa, código testável e soluções escaláveis com PostgreSQL, Docker e AWS.
